Welcome to the **Customer Segmentation in Marketing** project  
This project analyzes real onboarding survey data to identify **meaningful customer groups** based on:

- Acquisition channels  
- Geography  
- Engagement  
- Customer lifetime value (CLV)

The goal is to understand **which channels work best**, **which regions perform well**, and **where marketing investment should go**.
